Switchboard Privacy Policy

Switchboard is a Chrome extension that displays closer availability and a broadcast calling note inside Mojo. This policy explains what data the extension accesses and how it is handled.

What the extension accesses

  • Mojo page content: Switchboard injects a small UI bar into Mojo pages so users can see availability.
  • Availability and broadcast note: Switchboard requests status and note data from a configured endpoint (for example, a Google Apps Script URL).

What the extension does not do

  • Switchboard does not sell personal information.
  • Switchboard does not run ads or track users across websites.
  • Switchboard Lite does not write or modify lead data in Mojo.

Data storage

Switchboard may store basic settings locally in your browser using Chrome extension storage. Examples include UI preferences such as positioning or visibility toggles.

Where the extension runs

Switchboard is designed to run on Mojo pages (for example, domains ending in mojosells.com).

Third-party services

If your organization uses a third-party endpoint to provide availability or a broadcast note (such as Google Apps Script), that service processes the request and returns the data. Switchboard only displays the returned data.

Security

We aim to minimize data access. If you believe Switchboard is accessing more than it should, contact us and we will investigate.

Changes

We may update this policy as the product evolves. Updates will be posted on this page with a revised date.
